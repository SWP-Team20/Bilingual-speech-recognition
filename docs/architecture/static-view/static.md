# Static view assets

This directory contains the maintained source artifacts for the static architecture view.

- [`component-diagram.mmd`](component-diagram.mmd) — Mermaid source for the component diagram embedded in [`../README.md`](../README.md#static-view-component-diagram).

The static view explains the delivered system structure: React/Vite frontend, FastAPI backend, authentication and API routers, PostgreSQL metadata/index storage, backend filesystem artifact storage, and local speech-processing dependencies.

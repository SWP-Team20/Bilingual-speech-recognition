# Reflection

## Learning points
The team has learned:
* to create filters and search system;
* to create and describe architecture decisions.


## Validated assumptions
* Redesign was confirmed as successful;
* Filters, search, and admin panel works;
* Audio storage information displays as it is;
* Words in transcription are modifiable for future statistics.


## Friction and gaps
* For user role, transcription appears without spaces;
* User can enter two or more words to one word field while transcription patching, and it will be counted as one word;
* Speaker diarization is not final, system does not remember speakers by voice;
* Design should be consistent, so all alerts should be replaced with modals.

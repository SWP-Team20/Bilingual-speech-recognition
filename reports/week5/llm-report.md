# LLM Usage Report

This report documents the use of Large Language Model tools during the development of the Bilingual Speech Recognition project.

## Tools Used

The team used the following LLM-based tools:

- Cursor
- Claude
- Gemini

## Frontend Development

Cursor was used during frontend development to assist with the redesign of icons and restructuring of frontend code. It helped generate and improve implementation ideas, organize UI-related code, and support refactoring decisions.

Claude was also used for frontend-related assistance, including reviewing ideas, improving code structure, and helping reason about frontend implementation details.

All frontend changes were reviewed and adjusted by the team before being included in the project.

## Backend Development

Cursor and Claude were used during backend development to support implementation, debugging, and code-structure decisions. They helped the team reason about backend logic, improve maintainability, and check possible implementation approaches.

The team reviewed backend suggestions before applying them to the repository.

## Text and Documentation Review

Gemini was used to check written text for mistakes. This included reviewing documentation wording, grammar, and clarity.

Gemini was used only as a support tool for text correction. The final documentation content was reviewed by the team.

## Human Responsibility

LLM tools were used as development and writing assistants. The team remained responsible for reviewing, validating, editing, and approving all generated or suggested content before it was committed to the repository.
